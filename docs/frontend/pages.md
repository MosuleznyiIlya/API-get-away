# Pages

## Dashboard

**Purpose**: Главная страница с обзором метрик и состояния системы.

**Data Sources**:
- Metrics API — для графиков и KPI
- Services API — для статуса сервисов
- Logs API — для последних ошибок

**Actions**:
- Просмотр метрик за разные периоды (1h, 24h, 7d)
- Обновление данных (refresh)
- Переход к детальным отчетам

**Permissions**: Доступно всем аутентифицированным пользователям.

## Services

**Purpose**: Управление upstream-сервисами (CRUD).

**Data Sources**:
- Services API — список, создание, обновление, удаление

**Actions**:
- Создание нового сервиса
- Редактирование сервиса
- Удаление сервиса
- Просмотр деталей сервиса
- Health check

**Permissions**: Только администраторы.

## Routes

**Purpose**: Управление маршрутами проксирования (CRUD).

**Data Sources**:
- Routes API — список, создание, обновление, удаление
- Services API — для выбора upstream-сервиса

**Actions**:
- Создание маршрута
- Редактирование маршрута (path, method, priority)
- Удаление маршрута
- Тестирование маршрута
- Просмотр статистики по маршруту

**Permissions**: Только администраторы.

## API Keys

**Purpose**: Управление API ключами доступа.

**Data Sources**:
- API Keys API — список, создание, отзыв

**Actions**:
- Создание нового API Key (показ ключа один раз)
- Отзыв (revoke) API Key
- Просмотр статистики использования ключа
- Копирование ключа в буфер обмена

**Permissions**: Только администраторы.

## Logs

**Purpose**: Просмотр и фильтрация логов запросов.

**Data Sources**:
- Logs API — список логов с фильтрацией

**Actions**:
- Фильтрация по дате, сервису, маршруту, статусу
- Поиск по path, client_ip
- Экспорт в CSV/JSON
- Просмотр деталей лога (request/response)

**Permissions**: Доступно всем аутентифицированным пользователям (read-only).
